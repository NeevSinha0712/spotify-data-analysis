\# Spotify Music Data Analysis



\## Overview

This project performs Exploratory Data Analysis (EDA) on Spotify datasets.



\## Tools Used

\- Python

\- Pandas, NumPy

\- Matplotlib, Seaborn



\## Key Insights

\- Peak streaming era: 2016–2020

\- Pop dominates global charts

\- Weak correlation between danceability and streams
