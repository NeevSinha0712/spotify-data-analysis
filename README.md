# Spotify Music Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on multiple Spotify datasets, including All-Time Top 100 songs and Spotify Wrapped 2025 data, to uncover trends in popularity, genres, and audio features.

## Objectives
- Analyze trends in music popularity over time
- Study relationships between audio features and streams
- Identify dominant genres and artist patterns

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Insights
- Peak streaming era: 2016–2020
- Pop dominates global charts
- Weak correlation between danceability and streams
- Collaborations significantly increase reach

## Project Structure
- Data/ → datasets used
- Notebook/ → analysis code
- Report/ → detailed report
